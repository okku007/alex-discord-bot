## Requirements
- Node.js 18+ (discord.js v14 tested)
- npm

## Quick start
1) `git clone <repo>`  
2) `cd alex-discord-bot`  
3) `npm install`  
4) Copy `example-config.json` to `config.json`, then set:
   - `token`: your bot token from the Discord Developer Portal
   - `prefix`: command prefix (default `.`)
5) `npm start`
